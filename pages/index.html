# terminal

Full PTY emulation, xterm-256color, scrollback up to 50k lines. Metal GPU rendering. iPhone and iPad, portrait and landscape.

# auth

Secure Enclave keys with Face ID. Imported Ed25519, ECDSA, or RSA keys in Keychain. Passwords in Keychain. All biometric-protected.

# host verification

Trust-on-first-use. Warns on fingerprint change. QR code scanning for fingerprint import.

# input

On-device voice input via WhisperKit. No audio leaves your device. Keyboard toolbar with shortcut packs for Shell, Tmux, Vim, Claude Code.

# appearance

14 themes. 4 monospace fonts. Adjustable font size.

# privacy

No accounts. No analytics. No tracking. Everything on-device.
